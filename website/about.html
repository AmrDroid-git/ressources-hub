À propos

Un petit hub pour un grand objectif

Resources Hub a été créé pour regrouper les sites de ressources éducatives dans un seul endroit clair, simple et facile à partager.

L’idée

Après la création de plusieurs sites de ressources, il devient important d’avoir une page centrale qui guide les visiteurs vers le bon site. Ce hub évite la confusion : au lieu de partager plusieurs liens séparés, il suffit de partager ce site puis chacun choisit la plateforme qui l’intéresse.

Comment les sites sont ajoutés ?

Les sites affichés dans la page Sites web sont stockés dans un fichier JSON. Chaque élément contient le nom du projet, sa description, le lien officiel du site, le dépôt GitHub et le propriétaire. Cela permet d’ajouter de futurs projets rapidement sans refaire toute la structure du site.

Objectif

L’objectif est de rendre les ressources plus accessibles, mieux organisées et plus faciles à partager avec les étudiants. Le projet peut évoluer avec de nouvelles plateformes selon les besoins de la communauté.

Créateur

Les deux premiers sites listés dans ce hub ont été créés et maintenus par Amr Slama. Le hub garde la même logique : une interface propre, des liens directs, une structure simple et un style classique.
